# Contributing to MediVoice

## Development Setup from Mobile/Phone

Since you're developing from a phone, here's how to contribute:

### Option 1: GitHub Web Editor (Recommended)
1. Navigate to the file you want to edit
2. Click the pencil icon (✏️) to edit
3. Make changes and commit directly

### Option 2: GitHub Codespaces (If Available)
1. Click "Code" → "Codespaces" → "Create codespace"
2. Use the web-based VS Code editor

### Option 3: GitPod
1. Prefix the repo URL with `gitpod.io/#`
2. Example: `https://gitpod.io/#github.com/YOUR_USERNAME/medivoice`

## Commit Convention
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `style:` Formatting
- `refactor:` Code restructuring
- `test:` Adding tests
- `chore:` Maintenance

## Pull Request Process
1. Create a feature branch
2. Make changes
3. Test thoroughly
4. Submit PR with description

## Code Style
- Follow existing patterns
- Use TypeScript for frontend
- Use Python type hints for backend
- Keep functions small and focused
